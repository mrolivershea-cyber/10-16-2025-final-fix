ВАРИАНТ 1: Через Git (САМЫЙ ПРОСТОЙ) ⭐
----------------------------------------

1. Нажмите кнопку "Save to Github" в интерфейсе Emergent
2. Выберите или создайте репозиторий
3. Сохраните изменения
4. На вашем сервере:
   git clone YOUR_REPO_URL
   cd YOUR_REPO


ВАРИАНТ 2: Скачать архив вручную
---------------------------------

Файл: /app/pptp_socks_deployment.tar.gz

К сожалению, прямая ссылка для скачивания недоступна в этом интерфейсе.

Используйте Вариант 1 (Git) или Вариант 3 (копирование кода).


ВАРИАНТ 3: Ручное копирование кода (если Git недоступен)
--------------------------------------------------------

Я могу показать полный код каждого файла.
Вы скопируете код и создадите файлы вручную на вашем сервере.

ВАРИАНТ 4: Rollback на другом сервере
--------------------------------------

Если у вас есть доступ к функции Rollback на Emergent:
1. Сохраните текущее состояние как checkpoint
2. Затем восстановите его на вашем сервере
